# Gerador de QRCode 

## Descrição:
Este projeto é uma aplicação web que gera um QRCode a partir de um texto digitado pelo usuário

## Frameworks/Bibliotecas :
- [Vue.js](https://vuejs.org/)
- [QRious](https://github.com/neocotic/qrious)
